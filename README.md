# The Nine Secrets / 遮天九秘

基于小说《遮天》九秘的 Terraria tModLoader 模组。九秘者，临、兵、斗、者、皆、阵、列、前、行，每一秘都蕴含独特的力量。

## 功能特色

- **9个基础护符** - 每个拥有独特能力
- **5个双合一护符** - 融合两个基础护符的力量
- **3个三合一护符** - 更强大的组合能力
- **1个终极护符** - 九秘合一
- **互斥装备系统** - 同时只能装备一个护符
- **自定义合成站** - 秘术熔炉
- **Boss必掉护符** - 击败Boss获得对应护符
- **支持 Recipe Browser** - 合成表模组兼容

## 护符列表

### 基础护符 (Tier 1)

| 护符 | 效果 |
|------|------|
| **斗字印** | +8% 近战伤害，3% 额外打击几率 |
| **者字印** | +1 HP/s 生命回复，受伤时额外 +3 HP/s |
| **临字印** | +10% 防御，濒死保护（60秒冷却） |
| **兵字印** | +5% 全伤害，+25% 击退力 |
| **皆字印** | +5% 暴击率，+8% 物品掉落率 |
| **阵字印** | 召唤2枚环绕符文，造成 25 DPS |
| **列字印** | +13% 移动速度，+100% 飞行时间 |
| **行字印** | 10% 闪避几率，闪避后获得速度爆发 |
| **前字印** | 子弹时间 2.5 秒（120秒冷却），+5% 暴击伤害 |

### 合成配方

部分基础护符可通过合成获得：

- **斗字印**: 黑曜石 x30 + 狱石锭 x5 + 铁锭 x10 @ 铁砧
- **者字印**: 心形灯笼 + 再生手环 + 金锭 x10 @ 工匠作坊
- **临字印**: 钴蓝盾 + 黑曜石盾 + 铁锭 x15 @ 工匠作坊
- **皆字印**: 幸运马蹄铁 + 金锭 x10 + 琥珀 x5 @ 工匠作坊

## 注意事项

- 同时只能装备一个护符
- 合成护符需要在秘术熔炉中制作
- 终极护符需要三个高级护符和终极核心

## 项目结构

```
TheNineSecrets/
├── Items/
│   ├── Accessories/
│   │   ├── BaseSecrets/      # 基础护符
│   │   ├── Combined/         # 合成护符
│   │   ├── Tier3/            # 高级护符
│   │   └── Ultimate/         # 终极护符
│   └── Crafting/             # 合成材料
├── Players/                  # 玩家状态管理
├── NPCs/                     # Boss掉落配置
├── Projectiles/              # 弹幕（符文）
├── Tiles/                    # 秘术熔炉
├── Buffs/                    # 冷却Buff
├── Systems/                  # Recipe Browser支持
└── Localization/             # 本地化文件
```
